Необходимо создать веб-приложение, согласно условий:

1) внешне приложение представляет собой одну заглавную страницу проекта;
2) при открытии страницы:
- бекенд генерирует древовидную структуру в любом формате. "Дерево" должно иметь случайное количество узлов (например от 100 до 1000) и произвольную иерархию;
- фронтент (вью) отображает полученное дерево в виде вложенных ul списков. Каждый пункт полученного списка должен содержать номер узла и быть гиперссылкой;
3) при клике на гиперссылку из древовидных список должно открывать всплывающее окно любого стиля\формата, в котором:
- отображен виджет re-captcha с полем ввода кода, конпка "Продолжить" и кнопка закрытия окна;
- при вводе кода и нажатии на продолжить (в случае корректного ввода кода), бекенд делает запрос к API сервиса Giphy и получает случайное изображение;
- полученное изображение размещается на месте блока re-captcha;
- использование csrf токена обязательно;
- каждое открытие всплывающего окна должно показывать новый код re-captcha;
4) обязательное условие: использование Yii2.

Любые сторонние модули и расширения использовать разрешено.
В результате необходимо предоставить ссылку на проект в GitHub.
