Написать класс на PHP, на вход которому будет приходить: 
 - одномерный отсортированный по возрастанию массив натуральных чисел, содержащий N элементов
 - натуральное число K
 
На выходе он должен выдавать многомерный массив, который состоит из массивов, содержащих все комбинации элементов входящего массива и имеет размер N-K. При этом каждый такой массив должен остаться отсортированным.
 
Пример:
Входящий массив [1, 2, 3, 4]
K = 1
 
—
 
[1, 2, 3]
[1, 2, 4]
[1, 3, 4]
[2, 3, 4]
